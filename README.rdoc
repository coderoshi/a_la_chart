== a la Chart

This is a general framework for inserting various type of charting implementations - from grabbing the
data, to declaring how those values are mapped to the desired type of chart (pie, line, bar, etc).

== Example

To install this plugin:

    script/plugin install git://github.com/coderoshi/a_la_chart.git
